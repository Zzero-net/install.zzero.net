Getting Started

Quick Start

Three steps from zero to validator.

1

Install

Download and install the zero-node binary for your platform.

$ curl -sSf https://install.zzero.net/install.sh | sh
2

Initialize

Generate your validator keys and default configuration.

$ zero-node init --validator
3

Run

Start your node and begin syncing with the network.

$ zero-node run

Source

Build from Source

Requires Rust 1.75+ and Cargo.

$ git clone https://github.com/Zzero-net/zero-node $ cd zero-node $ cargo build --release

The compiled binary will be at target/release/zero-node. Copy it to your PATH.

Trust Score
Trust Scoring: Validators build trust over time based on uptime, correct attestations, and network participation. Higher trust scores receive proportionally greater block rewards. New validators start with a base trust score that increases after the first epoch (~24 hours).
